# Поиск законодательства РФ

## Назначение

Этот навык используется только для получения актуальных правовых источников через интернет.

Он не предназначен для проверки договора целиком. Передавай в поиск короткий обезличенный правовой вопрос.

## Обязательный вызов

Вызови инструмент `run_js`.

- script name: `index.html`
- data: JSON-строка со следующими полями:
  - `query`: строка до 400 символов;
  - `max_results`: целое число от 1 до 5, обычно 3;
  - `search_depth`: `basic` или `advanced`, обычно `advanced`.

Пример:

```json
{
  "query": "ГК РФ статья 395 проценты за пользование чужими денежными средствами актуальная редакция",
  "max_results": 3,
  "search_depth": "advanced"
}
```

## Правила запроса

- Не передавай полный договор.
- Не передавай ФИО, паспортные данные, реквизиты, коммерческую тайну или описание конкретного клиента.
- Сформулируй общий юридический вопрос.
- Для ГК РФ указывай номер статьи и предмет вопроса, если номер известен.
- Не вызывай действие `analyze`.
- Не вызывай `run_intent`.

## После получения результата

1. Используй только возвращённые официальные ссылки и фрагменты.
2. Для каждого правового вывода укажи источник и URL.
3. Не называй фрагмент полной точной цитатой, если результат содержит только поисковый отрывок.
4. Если результатов недостаточно, скажи, что точный текст нужно открыть и проверить на официальном портале.
5. Не скрывай, что поиск выполнен через Tavily.
