# Legal AI Edge Gallery Skills

Набор русскоязычных Agent Skills для Google AI Edge Gallery.

## Навыки

| Каталог | Назначение | Режим |
|---|---|---|
| `legal-text-simplifier` | Объяснение сложного юридического текста простыми словами | Офлайн |
| `legal-contract-auditor` | Проверка договора | Офлайн; нормы можно искать отдельно |
| `legal-template-adapter` | Адаптация шаблона | Офлайн; нормы можно искать отдельно |
| `legal-case-brief` | Конспект материалов дела | Офлайн; нормы можно искать отдельно |
| `legal-position-red-team` | Проверка слабых мест позиции | Офлайн; нормы можно искать отдельно |
| `legal-document-drafter` | Проекты юридических документов | Офлайн; нормы можно искать отдельно |
| `legal-law-version-comparator` | Сравнение двух редакций | Офлайн |
| `legal-calculator` | Расчёты через JavaScript | Офлайн после установки |
| `russian-law-search` | Поиск актуальных норм по официальным доменам | Интернет + Tavily API key |

## Главное изменение версии 3

Text-only навыки теперь прямо запрещают модели вызывать несуществующее действие `analyze`. После загрузки навык должен отвечать обычным текстом.

Для актуального законодательства добавлен отдельный `russian-law-search`. Он:

- вызывает только `run_js`;
- использует Tavily Search API;
- ограничивает выдачу официальными доменами;
- не передаёт договор в сеть, если модель соблюдает инструкции;
- требует отдельный API-ключ;
- не работает без интернета.

## Рекомендуемые комбинации

### Простое объяснение

Включите только:

```text
legal-text-simplifier
```

### Проверка договора без интернета

Включите:

```text
legal-contract-auditor
```

Передайте договор текстом. Для проверки конкретной нормы вставьте её текст вручную.

### Проверка договора с актуальными нормами

Включите одновременно:

```text
legal-contract-auditor
russian-law-search
```

Не включайте остальные навыки без необходимости.

## Изображения на iPhone

В iOS-версии Agent Skills обработка изображений с пользовательскими навыками может завершаться ошибкой выполнения действия. Надёжный порядок:

1. Распознайте изображение через **Ask Image** или системный Live Text.
2. Скопируйте полученный текст.
3. Создайте новый чат в **Agent Skills**.
4. Включите один нужный навык.
5. Вставьте текст вместо изображения.

Ошибка на скриншоте возникает после OCR: модель вызывает действие `analyze`, которого нет среди поддерживаемых инструментов. Обновлённые навыки явно запрещают такой вызов, но ошибка оболочки iOS при работе с изображениями всё равно может сохраняться.

## Публикация

Загрузите содержимое архива в корень репозитория и включите GitHub Pages:

```text
Settings → Pages → Deploy from a branch
Branch: main
Folder: / (root)
```

В приложение передавайте URL папки:

```text
https://alexandermint.github.io/legal-edge-gallery-skills/legal-text-simplifier/
```

а не URL файла `SKILL.md`.

## Проверка

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements-dev.txt
python tools/validate_skills.py
```

## Безопасность

`russian-law-search` отправляет короткий поисковый запрос и API-ключ в Tavily. Не передавайте в запрос полный договор, персональные данные или коммерческую тайну.

Поисковый фрагмент не заменяет проверку полного официального текста и редакции документа.
