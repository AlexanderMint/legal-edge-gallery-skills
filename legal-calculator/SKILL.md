# Юридический калькулятор

## Когда использовать

Используй навык для арифметической проверки или расчёта, когда пользователь предоставил исходные параметры.

Не определяй по памяти применимую ставку, начало просрочки, мораторий, лимит ответственности, рабочие дни или юридическую формулу.

## Обязательный вызов инструмента

Для расчёта вызови инструмент `run_js` со следующими параметрами:

- script name: `index.html`
- data: JSON-строка по одной из схем ниже.

Передавай денежные суммы и ставки строками, чтобы сохранить десятичную точность.

## Операции

### Простые годовые проценты

```json
{
  "operation": "simple_interest",
  "principal": "100000.00",
  "annual_rate_percent": "16.00",
  "days": 30,
  "year_basis": 365
}
```

Поля:

- `principal`: неотрицательная десятичная строка;
- `annual_rate_percent`: неотрицательная десятичная строка;
- `days`: целое неотрицательное число;
- `year_basis`: `365` или `366`.

### Договорная неустойка в процентах за день

```json
{
  "operation": "penalty_percent_per_day",
  "principal": "100000.00",
  "daily_rate_percent": "0.10",
  "days": 20,
  "cap": null
}
```

`cap` — десятичная строка или `null`.

### Суммирование требований

```json
{
  "operation": "sum_items",
  "items": [
    {"label": "Основной долг", "amount": "100000.00"},
    {"label": "Неустойка", "amount": "2500.50"}
  ]
}
```

### Календарные дни

```json
{
  "operation": "calendar_days",
  "start_date": "2026-01-01",
  "end_date": "2026-01-31",
  "inclusion": "exclude_start"
}
```

`inclusion`:

- `exclude_start` — начальный день не учитывается, конечный учитывается;
- `include_both` — учитываются оба дня;
- `exclude_both` — не учитываются оба дня.

Калькулятор считает только календарные дни в UTC. Он не знает праздники и переносы рабочих дней.

## После выполнения

1. Покажи исходные параметры.
2. Покажи использованную формулу.
3. Покажи результат до округления, если он возвращён.
4. Покажи итог в формате с двумя знаками после запятой.
5. Отдельно перечисли правовые предпосылки, которые не проверялись.
6. Не называй расчёт расчётом по конкретной статье закона, если применимость формулы не подтверждена пользователем.
