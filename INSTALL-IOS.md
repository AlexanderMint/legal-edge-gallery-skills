# Использование на iPhone

## Установка

Добавляйте навык по URL папки:

```text
https://alexandermint.github.io/legal-edge-gallery-skills/legal-text-simplifier/
```

## Ошибка `Failed to perform action (analyze)`

Эта ошибка означает:

1. навык был загружен;
2. модель получила текст;
3. модель попыталась вызвать неподдерживаемое действие `analyze`;
4. приложение не смогло выполнить это действие.

В обновлённых text-only навыках явно написано отвечать напрямую и не вызывать `analyze`.

## Надёжный сценарий для фотографии договора

1. Откройте **Ask Image**.
2. Попросите: `Распознай весь текст без анализа и сохрани структуру`.
3. Скопируйте результат.
4. Откройте новый чат **Agent Skills**.
5. Включите только `legal-text-simplifier` или `legal-contract-auditor`.
6. Вставьте распознанный текст.
7. Не прикладывайте изображение в этот же запрос.

## Актуальное законодательство

Без интернета:

- вставьте текст статьи или редакции закона вручную;
- основной навык использует этот текст как источник.

С интернетом:

- включите основной навык;
- включите `russian-law-search`;
- при первом вызове введите API-ключ Tavily;
- не передавайте в поиск весь договор.
